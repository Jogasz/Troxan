# Troxan

Troxan is a retro-inspired raycasting dungeon game built with `.NET 8` and `OpenTK`.

## Features
-Real-time raycasted world rendering
-Separate wall/floor/ceiling/sprite render passes
-Menu + HUD system
-Player stamina/combat/stat systems
-Local JSON settings and runtime stat persistence
-Optional API login and stat synchronization

## Tech Stack
-`C#`
-`.NET 8`
-`OpenTK`
-`StbImageSharp`

## Run Locally
1. Open the project in Visual Studio.
2. Ensure `.NET 8 SDK` is installed.
3. Build and run `Troxan.csproj`.

## Project Structure
-`source/` core gameplay, engine, rendering logic
-`assets/` maps, textures, and game resources
-`documents/` project documentation
